Guida Ruby on rails

Comando Descrizione
rails new "app name" Serve a creare una nuova app con rails
rails server Serve ad avviare il server rails
rails g controller "controller name" "action name" Serve a generare un controller e un'azione
root to: "controllername#actionname" Serve per impostare l'homepage della nostra applicazione
link_to "parola_visualizzata" ,path/url Serve a creare dei collegamenti ipertestuali(link)
rails g scaffold "scaffold_name" "istruzoni" Serve a creare un intera risorsa web dalle liste ai controlli ai database
